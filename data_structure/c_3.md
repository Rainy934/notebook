# 二叉树
> 每个节点都存在不多于两个子节点

### 二叉树的实现
因为节点固定所以可直接用指针指向

    function TreeNode () {
        this.left = null
        this.right  = null
    }

### 二叉树的应用
编译器的设计领域
1. 表达式树的应用，先序遍历，中序遍历， 后序遍历
2. 查找应用

### 实践
